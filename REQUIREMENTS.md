# Требования: Викторина «Какая твоя кофейная личность?»

## Обзор
Викторина для программы лояльности Basecamp Coffee. Клиенты отвечают на вопросы, получают кофейную личность и рекомендацию напитка.

## Пары личность → кофе (6 штук)

| # | Личность | Напиток | Теглайн |
|---|----------|---------|---------|
| 1 | Уютная Классика | Средняя Обжарка Дрип | «Комфорт в каждой чашке» |
| 2 | Дзен Минималист | Чёрный кофе, Одиночное Происхождение | «Просто. Чисто. Идеально.» |
| 3 | Ночная Сова | Красный Глаз (кофе + эспрессо) | «Сон опционален» |
| 4 | Здоровяк | Американо с Овсяным Молоком | «Здоровье в каждом глотке» |
| 5 | Артист Сноб | Пуровер, Одиночное Происхождение | «Ты знаешь, что нравится» |
| 6 | Практичный Прагматик | Большой Дрип, Свежий что есть | «Просто пусть работает» |

## Формат результата
**Вариант A** — одна рекомендация: «Ты — Уютная Классика! Попробуй нашу Среднюю Обжарку Дрип.»

## Визуальный стиль
Тёплый/уютный: землистые тона, мягкие градиенты, ощущение кофейни.

## Формат ответов
Карточки с цветным фоном — каждый ответ как мини-карточка.

## Изображения
Да — фото напитков для каждого результата.

## Иконки
Нет отдельных иконок — карточки с фоном визуально достаточно.

## Вопросы викторины (7 штук)

### Вопрос 1: Идеальные выходные?
- А) Прогулка по книжным магазинам и кофейням → Уютная Классика
- Б) Тишина, медитация, никаких планов → Дзен Минималист
- В) Выход в бар или на вечеринку до утра → Ночная Сова
- Г) Поход, пробежка или новый фитнес-класс → Здоровяк
- Д) Музей, галерея, что-то эстетичное → Артист Сноб
- Е) Разобрать шкаф, починить кран, план на неделю → Прагматик

### Вопрос 2: Твой любимый жанр фильма?
- А) Комфортное кино, которое уже видел → Уютная Классика
- Б) Документалки, что-то осмысленное → Дзен Минималист
- В) Хоррор или триллер ночью → Ночная Сова
- Г) Спортивные фильмы, мотивация → Здоровяк
- Д) Арт-хаус, авторское кино → Артист Сноб
- Е) Документалки о бизнесе, «как это работает» → Прагматик

### Вопрос 3: Какой цвет тебе ближе?
- А) Тёплый бежевый, кремовый → Уютная Классика
- Б) Спокойный зелёный → Дзен Минималист
- В) Тёмно-фиолетовый → Ночная Сова
- Г) Свежий мятный → Здоровяк
- Д) Глубокий бордовый → Артист Сноб
- Е) Серый, практичный → Прагматик

### Вопрос 4: Твой идеальный завтрак?
- А) Овсянка с корицей и свежий хлеб → Уютная Классика
- Б) Просто вода и лёгкий салат → Дзен Минималист
- В) Завтрак? Я только лёг спать → Ночная Сова
- Г) Смузи с протеином и зеленью → Здоровяк
- Д) Авокадо-тост в стильном кафе → Артист Сноб
- Е) Что угодно, быстро, без заморочек → Прагматик

### Вопрос 5: Ты в книжном магазине — тянешься к...
- А) Классическая литература, проверенные авторы → Уютная Классика
- Б) Философия, книги о mindfulness → Дзен Минималист
- В) Триллеры, детективы, что-то с напряжением → Ночная Сова
- Г) Здоровье, спорт, биографии атлетов → Здоровяк
- Д) Искусство, дизайн, фотокниги → Артист Сноб
- Е) Бизнес, продуктивность, «как заработать» → Прагматик

### Вопрос 6: На необитаемом острове берёшь с собой...
- А) Плед и хорошую книгу → Уютная Классика
- Б) Ничего. Наслаждаюсь тишиной → Дзен Минималист
- В) Колонку и плейлист → Ночная Сова
- Г) Спортивный инвентарь → Здоровяк
- Д) Скетчбук и карандаши → Артист Сноб
- Е) Нож и набор для выживания → Прагматик

### Вопрос 7: Идеальный напиток (не кофе)?
- А) Горячий какао с маршмеллоу → Уютная Классика
- Б) Травяной чай → Дзен Минималист
- В) Энергетик или крепкий чай → Ночная Сова
- Г) Зелёный смузи или кефир → Здоровяк
- Д) Крафтовый коктейль с необычными нотами → Артист Сноб
- Е) Вода. Зачем усложнять? → Прагматик

## Логика подсчёта
Ответы = голоса за личности. В конце подсчитываем, какая личность набрала больше всего голосов. Показываем одну рекомендацию.
